Managing team assets

In the backoffice team, click on the Team assets entry, on the left, you should see the list of existing assets

Warning

If a Something went wrong !!! message is displayed, look a the Tenant / Bucket configuration

Add a new assets

Select a file after click toh the Select file button, then the filename, title and content-type should be pre-filled. You can overwrite these informations and add a description if you want. Click to the Add asset button to finalize the action. This asset can be just used for this team.
