• Developper portal
  • for Otoroshi

Daikoku is the missing developper portal for Otoroshi written in Scala and developped by the MAIF OSS team. Daikoku is designed to manage your apis with ad hoc consumption plans and self service usage.

Features

  • Gorgeous admin UI
  • Rest admin API
  • Event-Driven
  • Official docker image
  • Available as single jar file

easy to configure

No config files, gorgeous admin UI, slick admin API. Live reconfiguration at runtime

high productivity

Freedom to chose your own stack, no library needed. Route and secure microservices in matter of seconds and accelerate your time to market
